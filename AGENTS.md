# paradise_site

## ¿Qué es?
Sitio web de alquiler vacacional de lujo para "Paradise House Miami" — propiedad de 6 habitaciones en Cutler Bay, FL.

## Stack
- Vanilla HTML + CSS + JavaScript (sin build tools)
- SEO con Schema markup (LodgingBusiness, FAQPage)
- Integración WhatsApp para reservas directas

## Archivos
```
index.html
css/styles.css
js/main.js
```

## Diseño
- Tema oscuro de lujo: tonos tierra oscuros (#1A0900+)
- Paleta verde azulado + aqua
- Fuentes: Cormorant Garamond (serif) + DM Sans

## Amenidades destacadas
- Piscina privada, sala de juegos, cancha de pickleball, mini golf, fire pit

## Estado actual

- SEO optimizado para búsqueda local (Cutler Bay, Miami)
